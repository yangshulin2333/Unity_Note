#### Lua/Luau 的设计哲学是
**一个运算符只对应一种语义**，绝不做隐式类型转换。

#### do
`do` 是 Luau 里的 **「代码块开关」**：
- 循环、条件、函数里，它是语法强制的「执行体入口」
- 手动写 `do...end`，是「隔离作用域、避免变量冲突」的神器
- 永远和 `end` 配对，顺序决定代码块边界
	- `function`、`if`、`for`、`while`、`repeat` **本身就代表代码块开始**，所以**不用写 do**。

在 Lua 语言一共有 **8 种基本数据类型**

| 数据类型     | 英文名称       | 描述与关键特性                                                                                                                                                                                                                                             |
| -------- | ---------- | --------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| **空**    | `nil`      | 表示“无效值”或“什么都没有”。**未赋值的变量默认就是 `nil`**；将全局变量或表中的元素赋值为 `nil` 等同于删除它[](https://developer.aliyun.com/article/1087356)[](https://m.w3cschool.cn/lua/lua-data-types.html)。                                                                                 |
| **布尔**   | `boolean`  | 只有两个值：`true`（真）和 `false`（假）。注意：在 Lua 的条件判断中，只有 `false` 和 `nil` 视为“假”，**数字 0 和空字符串都视为“真”**[](https://developer.aliyun.com/article/1087356)[](https://m.w3cschool.cn/lua/lua-data-types.html)。                                                        |
| **数值**   | `number`   | 用于表示所有的数字（包括整数和浮点数），**内部默认是双精度浮点数（double）**[](https://developer.aliyun.com/article/1087356)[](https://www.tencentcloud.com/techpedia/101878)[](https://doc.yonyoucloud.com/doc/wiki/project/lua/data-types.html)。                                   |
| **字符串**  | `string`   | 文本数据。可以用单引号 `' '`、双引号 `" "` 或双层方括号 `[[ ]]`（用于多行文本）表示[](https://m.w3cschool.cn/lua/lua-data-types.html)[](https://bbs.huaweicloud.com/blogs/478c09171e094f72940b06544dc21b86)。用 `..` 进行拼接，用 `#` 获取长度。                                                |
| **函数**   | `function` | 在 Lua 中，函数被视为“第一类值”，这意味着你可以**将函数存储在变量中**，或作为参数传递给其他函数[](https://developer.huaweicloud.com/tags/200539/post_1)[](https://m.w3cschool.cn/lua/lua-data-types.html)。                                                                                    |
| **表**    | `table`    | **这是 Lua 中唯一的数据结构工具，功能极其强大**。它可以被当作数组、字典、对象、模块等来使用。数组的**默认索引从 1 开始**，而不是 0[](https://developer.huaweicloud.com/tags/200539/post_1)[](https://developer.aliyun.com/article/1087356)[](https://m.w3cschool.cn/lua/lua-data-types.html)。               |
| **线程**   | `thread`   | 主要用于实现**协程（coroutine）**，这是一种轻量级的并发机制，允许一个程序有多个执行流，但同一时间只有一个在运行[](https://doc.yonyoucloud.com/doc/wiki/project/lua/data-types.html)[](https://m.w3cschool.cn/lua/lua-data-types.html)。                                                               |
| **用户数据** | `userdata` | 允许将任意的 **C/C++ 数据结构** 存储在 Lua 变量中。普通 Lua 脚本只能使用它（通常由宿主程序提供），不能直接创建或修改它[](https://developer.huaweicloud.com/tags/200539/post_1)[](https://m.w3cschool.cn/lua/lua-data-types.html)[](https://courses.cs.umbc.edu/331/fall15/05/notes/Lecture06.html)。 |

|数据类型|类型分类|赋值行为|
|---|---|---|
|`nil`, `boolean`, `number`, `string`|**值类型**|复制值，独立存储|
|`table`, `function`, `thread`, `userdata`|**引用类型**|复制引用，共享对象|
由于 Lua 中**只有 table 一种数据结构**，且函数、线程也是引用类型，因此**几乎所有复杂数据都是引用类型**。这既是 Lua 简单设计的体现，也是初学者容易犯错的地方——复制 table 时必须手动实现深拷贝，简单的 `t2 = t1` 只是共享同一份数据。

所有表函数，第一个参数 t 永远 = 你要操作的表！

### 模块脚本 = 表 + 函数挂表 + 返回表

lua

```
local 模块 = {}

end

return 模块
```

lua

```
local 模块 = require(路径)
模块.函数名()
```

### 通信对象
一个 RemoteEvent 一创建，**自带 4 个官方方法**：

#### 客户端能调用的：

- **`FireServer()`** → 客户端发给服务端

#### 服务端能调用的：

- **`FireClient()`** → 服务端发给某个玩家
- **`FireAllClients()`** → 服务端发给所有人
- **`FireClientAtPosition()`** → 发给附近玩家

FireServer 是官方方法
RemoteEvent 是官方对象
OnServerEvent 是官方事件：客户端发送消息过来时 运行
	OnserverEvent是RemoteEvent对象的内置事件，当客户端调用：FireServer()时，服务端会触发此事件

#### RemoteEvent

lua

```
-- 发
RemoteEvent:FireServer(data)

-- 收
RemoteEvent.OnServerEvent:Connect(function(player, data)
end)
```

#### RemoteFunction

lua

```
-- 发并等待返回
local result = RemoteFunction:InvokeServer(data)

-- 收并返回值
RemoteFunction.OnServerInvoke = function(player, data)
    return "服务器返回的结果"
end
```

### **RemoteNames 这个 ModuleScript = 一个万能容器**

它里面 **return 什么**，你 `require` 拿到的 **就是什么**！

---

#### 1. return 表

lua

```
local data = { name = "张三" }
return data
```

→ 拿到的就是 **表 / 字典**

#### 2. return 字符串

lua

```
return "我是文本"
```

→ 拿到的就是 **字符串**

#### 3. return 函数

lua

```
local function func() end
return func
```

→ 拿到的就是 **函数**

#### 4. return 数字

lua

```
return 100
```

→ 拿到的就是 **数字**
