
# **游戏功能模块表（即时战略 + 角色养成）**

|模块类别|功能名称|说明|优先级|免费资源/初期实现建议|
|---|---|---|---|---|
|**角色系统**|角色模型|玩家可操作的角色，带骨骼Rig，便于动画|高|Roblox Library 免费 R15/R6 模型，或 Mixamo Rigged FBX|
||属性系统|HP、攻击力、防御力、移动速度、技能冷却等|高|Script 自定义属性表|
||技能系统|主动技能（攻击/范围伤害/增益减益）+被动技能|高|初期可用简单 ParticleEmitter + Script 表现技能特效|
||装备系统|武器、防具、饰品，提升属性或技能效果|中|先用 Part 或简单 Mesh 代替装备模型|
||成长系统|经验值、等级提升、技能解锁|高|Script 记录经验值和等级|
|**战斗系统**|单位操控|玩家移动、攻击、技能释放|高|使用 Roblox Humanoid + Script 控制|
||队伍管理|小规模单位编队，AI控制非玩家单位|中|初期可先只控制玩家单位，AI用简单路径寻路|
||战斗地图|地形、障碍物、战斗区划分|高|Baseplate/Flat Terrain + Part + 免费模型|
||战斗奖励|经验、金币、装备掉落|高|Script 控制掉落与奖励发放|
|**大厅系统**|玩家入口|Spawn点，角色进入游戏|高|Starting Place 模板自带 SpawnLocation|
||商店|购买装备、道具、皮肤|中|UI + Script，图标可用免费 PNG/GUI Kit|
||任务/副本板|显示可挑战副本或战斗任务|中|UI + Script 简单任务列表|
||装备/角色养成界面|升级装备、查看角色状态、技能加点|中|GUI + Script，初期可简化|
|**UI 系统**|状态栏|显示 HP、技能冷却、金币等|高|Roblox GUI 或 Free GUI Kit|
||技能按钮|点击释放技能|高|GUI + Script + ParticleEmitter 表现技能|
||提示/通知|战斗消息、奖励提示|中|TextLabel + Script|
|**音效与特效**|战斗音效|攻击、技能、死亡音效|中|Roblox Library 免费音效 / Freesound|
||背景音乐|大厅、战斗区域循环音乐|中|Roblox Library / Free Music|
||特效|技能特效、击中效果|中|ParticleEmitter + MeshPart|
|**数据与存储**|玩家数据保存|等级、装备、金币、技能解锁|高|DataStoreService（Roblox 官方）|
||临时战斗数据|战斗中单位状态、位置|高|Script + Table 管理|
|**扩展模块**|多场景战斗|大厅和战斗区分开场景|低|TeleportService 管理场景切换|
||AI 敌人|自主移动、攻击、技能使用|中|PathfindingService + Script|
||多人同步|多玩家实时战斗|中|RemoteEvent + RemoteFunction|

---

💡 **开发建议**：

1. **初期原型**：
    
    - 优先实现 **角色系统、战斗系统、单场景大厅 + 战斗区**
        
    - 用 Roblox 免费资源代替复杂模型
        
2. **中期优化**：
    
    - 添加技能特效、装备系统、UI美化
        
    - 角色成长与任务系统完善
        
3. **后期扩展**：
    
    - 多场景管理、AI敌人、多人同步战斗
