# 先讲核心设计思想

在你真正动手前，先把思路吃透。

---

## 一、什么是 Config 表

Config 表就是“游戏规则表”。

比如建筑升级，不应该把规则写死在代码里像这样：

```lua
if buildingType == "Farm" and level == 1 then
	cost = 100
elseif buildingType == "Farm" and level == 2 then
	cost = 200
end
```

这样很难维护。

正确做法是把规则放到一个 ModuleScript 配置里：

```lua
Farm = {
    [1] = { UpgradeCost = { Coins = 100 } },
    [2] = { UpgradeCost = { Coins = 200 } }
}
```

这样你的代码只负责“读取规则”和“执行逻辑”。

这就是**数据驱动**。

---

## 二、为什么升级必须服务端做

因为升级会影响：
- 玩家货币
- 玩家建筑等级
- 后续资源产量
- 玩法解锁

这些都属于“核心收益数据”。
如果放客户端做，玩家可以轻松改本地脚本作弊。

所以：

- **客户端**：点击按钮、显示界面、请求升级
- **服务端**：检查条件、扣钱、改等级、保存数据、同步结果

---

## 三、什么是懒结算资源

懒结算就是：

> 不每秒真的给玩家加资源，而是在玩家点击收取时，一次性按时间差计算。

举例：

- 农田 1 级每分钟产 10 粮草
- 上次收取时间是 12:00
- 现在是 12:05
- 时间差是 5 分钟
- 理论产出是 50 粮草
- 如果容量上限是 40，那就只给 40

这样比每秒更新一次省性能太多。

---

## 四、为什么建筑模型和建筑数据要分离

建筑数据是：
- 等级
- 资源产量
- 上次收取时间

建筑模型是：
- 哪个模型显示
- 什么材质
- 什么外观

如果把它们绑死，会很难维护。
正确方式是：

- 数据存在 Profile
- 显示根据数据决定

即：
> 数据是“真相”，模型只是“表现”。

## BaseService 完整第二阶段骨架

```lua
local Workspace = game:GetService("Workspace")
local ReplicatedStorage = game:GetService("ReplicatedStorage")
local ServerStorage = game:GetService("ServerStorage")

local BuildingConfig = require(ReplicatedStorage:WaitForChild("Shared"):WaitForChild("Config"):WaitForChild("BuildingConfig"))

local BaseService = {}
BaseService.AssignedPlots = {}

local plotsFolder = Workspace:WaitForChild("Plots")
local baseTemplatesFolder = ServerStorage:WaitForChild("Templates"):WaitForChild("Bases")

function BaseService:GetAllPlots()
	return plotsFolder:GetChildren()
end

function BaseService:GetPlotByPlayer(player)
	return self.AssignedPlots[player]
end

function BaseService:IsPlotFree(plot)
	for _, assignedPlot in pairs(self.AssignedPlots) do
		if assignedPlot == plot then
			return false
		end
	end
	return true
end

function BaseService:AssignPlot(player)
	if self.AssignedPlots[player] then
		return self.AssignedPlots[player]
	end

	for _, plot in ipairs(self:GetAllPlots()) do
		if self:IsPlotFree(plot) then
			self.AssignedPlots[player] = plot
			print("[BaseService] Assigned", plot.Name, "to", player.Name)
			return plot
		end
	end

	warn("[BaseService] No free plot for", player.Name)
	return nil
end

function BaseService:ReleasePlot(player)
	local plot = self.AssignedPlots[player]
	if plot then
		print("[BaseService] Released", plot.Name, "from", player.Name)
		self.AssignedPlots[player] = nil
	end
end

function BaseService:GetPlotSpawn(plot)
	if not plot then
		return nil
	end
	return plot:FindFirstChild("BaseSpawn")
end

function BaseService:GetBuildingNode(plot, buildingType)
	if not plot then
		return nil
	end

	local buildNodes = plot:FindFirstChild("BuildNodes")
	if not buildNodes then
		return nil
	end

	return buildNodes:FindFirstChild(buildingType .. "Node")
end

function BaseService:GetBuildingData(profile, buildingType)
	if not profile or not profile.Base then
		return nil
	end
	return profile.Base[buildingType]
end

function BaseService:GetBuildingConfig(buildingType)
	return BuildingConfig[buildingType]
end

function BaseService:GetBuildingLevel(profile, buildingType)
	local buildingData = self:GetBuildingData(profile, buildingType)
	if not buildingData then
		return 1
	end
	return buildingData.Level or 1
end

function BaseService:GetCurrentLevelConfig(buildingType, level)
	local config = self:GetBuildingConfig(buildingType)
	if not config then
		return nil
	end
	return config.Levels[level]
end

function BaseService:GetNextLevelConfig(buildingType, currentLevel)
	local config = self:GetBuildingConfig(buildingType)
	if not config then
		return nil
	end
	return config.Levels[currentLevel + 1]
end

function BaseService:IsAtMaxLevel(profile, buildingType)
	local config = self:GetBuildingConfig(buildingType)
	if not config then
		return true
	end

	local level = self:GetBuildingLevel(profile, buildingType)
	return level >= config.MaxLevel
end

function BaseService:CanUpgradeBuilding(profile, buildingType)
	local config = self:GetBuildingConfig(buildingType)
	if not config then
		return false, "建筑不存在"
	end

	local currentLevel = self:GetBuildingLevel(profile, buildingType)
	if currentLevel >= config.MaxLevel then
		return false, "已达到最高等级"
	end

	local nextLevel = currentLevel + 1
	local nextConfig = config.Levels[nextLevel]
	if not nextConfig then
		return false, "下一等级配置不存在"
	end

	if buildingType ~= "MainHall" then
		local mainHallLevel = self:GetBuildingLevel(profile, "MainHall")
		local requirement = config.MainHallRequirement and config.MainHallRequirement[nextLevel]
		if requirement and mainHallLevel < requirement then
			return false, "主殿等级不足"
		end
	end

	local cost = nextConfig.UpgradeCost or {}
	for currencyType, amount in pairs(cost) do
		local currentAmount = profile.Currency[currencyType] or 0
		if currentAmount < amount then
			return false, currencyType .. "不足"
		end
	end

	return true, "可以升级"
end

function BaseService:PayUpgradeCost(profile, buildingType)
	local currentLevel = self:GetBuildingLevel(profile, buildingType)
	local nextConfig = self:GetNextLevelConfig(buildingType, currentLevel)
	if not nextConfig then
		return false
	end

	local cost = nextConfig.UpgradeCost or {}
	for currencyType, amount in pairs(cost) do
		profile.Currency[currencyType] = (profile.Currency[currencyType] or 0) - amount
	end

	return true
end

function BaseService:UpgradeBuilding(profile, buildingType)
	local canUpgrade, reason = self:CanUpgradeBuilding(profile, buildingType)
	if not canUpgrade then
		return false, reason
	end

	local buildingData = self:GetBuildingData(profile, buildingType)
	if not buildingData then
		return false, "建筑数据不存在"
	end

	self:PayUpgradeCost(profile, buildingType)
	buildingData.Level += 1

	return true, "升级成功"
end

function BaseService:GetResourceBuildingStoredAmount(profile, buildingType)
	local buildingData = self:GetBuildingData(profile, buildingType)
	local buildingConfig = self:GetBuildingConfig(buildingType)

	if not buildingData or not buildingConfig then
		return 0
	end

	if buildingConfig.Type ~= "Resource" then
		return 0
	end

	local level = buildingData.Level
	local levelConfig = buildingConfig.Levels[level]
	if not levelConfig then
		return 0
	end

	local lastCollectTime = buildingData.LastCollectTime or os.time()
	local currentTime = os.time()
	local elapsed = math.max(0, currentTime - lastCollectTime)

	local outputPerMinute = levelConfig.OutputPerMinute or 0
	local capacity = levelConfig.Capacity or 0

	local generated = (elapsed / 60) * outputPerMinute
	return math.floor(math.min(generated, capacity))
end

function BaseService:CollectBuildingResource(profile, buildingType)
	local buildingData = self:GetBuildingData(profile, buildingType)
	local buildingConfig = self:GetBuildingConfig(buildingType)

	if not buildingData or not buildingConfig then
		return false, "建筑不存在", 0
	end

	if buildingConfig.Type ~= "Resource" then
		return false, "该建筑不可收取资源", 0
	end

	local amount = self:GetResourceBuildingStoredAmount(profile, buildingType)
	if amount <= 0 then
		return false, "当前没有可收取资源", 0
	end

	local resourceType = buildingConfig.ResourceType
	profile.Currency[resourceType] = (profile.Currency[resourceType] or 0) + amount
	buildingData.LastCollectTime = os.time()

	return true, "收取成功", amount
end

function BaseService:ClearBuildingVisual(plot, buildingType)
	local node = self:GetBuildingNode(plot, buildingType)
	if not node then
		return
	end

	for _, child in ipairs(node:GetChildren()) do
		if child:IsA("Model") and child.Name == "CurrentBuildingModel" then
			child:Destroy()
		end
	end
end

function BaseService:RefreshBuildingVisual(plot, buildingType, level)
	local templatesFolder = baseTemplatesFolder:FindFirstChild(buildingType)
	if not templatesFolder then
		warn("[BaseService] Missing template folder for", buildingType)
		return
	end

	local template = templatesFolder:FindFirstChild("Level_" .. tostring(level))
	if not template then
		warn("[BaseService] Missing template for", buildingType, "level", level)
		return
	end

	local node = self:GetBuildingNode(plot, buildingType)
	if not node then
		return
	end

	self:ClearBuildingVisual(plot, buildingType)

	local clone = template:Clone()
	clone.Name = "CurrentBuildingModel"
	clone.Parent = node

	if clone.PrimaryPart then
		clone:PivotTo(node.CFrame + Vector3.new(0, clone.PrimaryPart.Size.Y / 2, 0))
	else
		clone:PivotTo(node.CFrame)
	end
end

function BaseService:RefreshAllBuildingVisuals(player, profile)
	local plot = self:GetPlotByPlayer(player)
	if not plot then
		return
	end

	for buildingType, buildingData in pairs(profile.Base) do
		local level = buildingData.Level or 1
		self:RefreshBuildingVisual(plot, buildingType, level)
	end
end

function BaseService:GetBuildingViewData(profile, buildingType)
	local config = self:GetBuildingConfig(buildingType)
	local buildingData = self:GetBuildingData(profile, buildingType)

	if not config or not buildingData then
		return nil
	end

	local level = buildingData.Level or 1
	local currentLevelConfig = self:GetCurrentLevelConfig(buildingType, level)
	local nextLevelConfig = self:GetNextLevelConfig(buildingType, level)
	local storedAmount = 0

	if config.Type == "Resource" then
		storedAmount = self:GetResourceBuildingStoredAmount(profile, buildingType)
	end

	return {
		BuildingType = buildingType,
		DisplayName = config.DisplayName,
		Type = config.Type,
		Level = level,
		MaxLevel = config.MaxLevel,
		CurrentConfig = currentLevelConfig,
		NextConfig = nextLevelConfig,
		StoredAmount = storedAmount,
		ResourceType = config.ResourceType,
	}
end

return BaseService
```

# 第 9 步：给 Plot 节点绑定交互脚本

现在玩家靠近建筑要能触发打开建筑面板。

最简单稳定的做法：
在服务端监听每个 Plot 中的 `ProximityPrompt.Triggered`

## 新建脚本
在：

`ServerScriptService > Systems`

下新建 Script：

`PlotPromptBinder.server`

填入代码：

```lua
local Players = game:GetService("Players")
local Workspace = game:GetService("Workspace")

local PlayerService = require(game.ServerScriptService.Services:WaitForChild("PlayerService"))
local BaseService = require(game.ServerScriptService.Services:WaitForChild("BaseService"))

local plotsFolder = Workspace:WaitForChild("Plots")

local function nodeNameToBuildingType(nodeName)
	return string.gsub(nodeName, "Node", "")
end

local function bindPrompt(prompt, node)
	prompt.Triggered:Connect(function(player)
		local plot = BaseService:GetPlotByPlayer(player)
		if not plot then
			return
		end

		if not node:IsDescendantOf(plot) then
			return
		end

		local buildingType = nodeNameToBuildingType(node.Name)
		PlayerService:OpenBuilding(player, buildingType)
	end)
end

local function scanPlot(plot)
	local buildNodes = plot:FindFirstChild("BuildNodes")
	if not buildNodes then
		return
	end

	for _, node in ipairs(buildNodes:GetChildren()) do
		local prompt = node:FindFirstChildOfClass("ProximityPrompt")
		if prompt then
			bindPrompt(prompt, node)
		end
	end
end

for _, plot in ipairs(plotsFolder:GetChildren()) do
	scanPlot(plot)
end
```

---

## 学习重点
这段代码的逻辑是：

1. 扫描所有 Plot
2. 扫描每个 Plot 的所有建筑节点
3. 找到节点上的 ProximityPrompt
4. 当玩家触发时，检查这个节点是不是玩家自己的 Plot
5. 是的话就打开建筑面板

这个“检查是否属于自己 Plot”很重要。
否则玩家会操作别人的建筑。

# 第 18 步：你在第二阶段真正学到了什么

这部分非常重要。你说你想“彻底掌握”，所以我明确告诉你这一阶段本质上你学的是什么。

---

## 你学到的 Roblox 核心知识点

### 1. ModuleScript 拆分职责
- `DataService` 管数据
- `BaseService` 管建筑
- `PlayerService` 管玩家生命周期
- `UIController` 管界面
- `BuildingController` 管建筑 UI 交互

### 2. 客户端/服务端职责分离
- 客户端只请求
- 服务端做判断
- 服务端发回结果

### 3. 数据驱动开发
- 建筑升级规则不写死
- 所有规则来自 `BuildingConfig`

### 4. 状态同步
