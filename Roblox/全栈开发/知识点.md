## UI
- **AnchorPoint：我以自己哪个点定位**
- **Position：我离父容器多远**
- **Size：我多大**
- **Size 的 offset = 变大**
- **Position 的 offset = 移动**
- 左上角固定写法：

plaintext

```
AnchorPoint (0,0)
Position {0,20, 0,20}
Size {0,300, 0,120}
```

## 想做「自适应 + 贴角落」

1. 角落方向 → 设置 `AnchorPoint`
2. 留一点边距 → `Position = {0/1, 20, 0/1, 20}`
	- 贴 **左边** → X 用 `0`
	- 贴 **右边** → X 用 `1`
	- 贴 **上边** → Y 用 `0`
	- 贴 **下边** → Y 用 `1`
3. 占屏幕比例 → `Size = {0.xx, 0, 0.xx, 0}`

## 想做「自适应 + 居中」

- AnchorPoint = `(0.5, 0.5)`
- Position = `(0.5,0, 0.5,0)`
- Size = `(0.xx,0, 0.xx,0)`
