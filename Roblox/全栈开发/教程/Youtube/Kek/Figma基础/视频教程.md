建议你按“够用主义”选教程：**只学 Figma 的 UI 制作 + PNG 导出 + Roblox Studio 还原**，暂时不要学完整 UI/UX 职业设计课程。

**我推荐的学习顺序**

1. **Figma 官方入门课**
   [Figma Design for beginners 2025](https://help.figma.com/hc/en-us/articles/30848209492887-Course-overview-Figma-Design-for-beginners-2025)

   只看这些部分：
   - Shapes / Text / Frames
   - Layers
   - Auto Layout
   - Components
   - Export

   不用深入学 Portfolio、完整网站、复杂原型。你是做 Roblox，不是找 UI/UX 岗。

2. **Figma 官方 Auto Layout**
   [Guide to Auto Layout](https://help.figma.com/hc/en-us/articles/360040451373-Guide-to-auto-layout)

   这个很重要。你做 Roblox UI 时会经常做：
   - 按钮组
   - 商店列表
   - 背包格子
   - 横向菜单
   - 自动适配文字长度的按钮

   Figma 的 Auto Layout 对应到 Roblox 里，你可以理解成类似：
   - `UIListLayout`
   - `UIGridLayout`
   - `UIPadding`

3. **Figma 导出设置**
   [Export formats and settings](https://help.figma.com/hc/en-us/articles/13402894554519-Export-formats-and-settings)

   你重点掌握：
   - PNG 导出
   - 1x / 2x 导出
   - 图层命名
   - 只导出选中元素
   - 透明背景

   Roblox UI 素材通常优先导出 PNG，不用 JPG。

4. **Roblox 官方 UI 课程**
   [Roblox User Interface Design Curriculum](https://create.roblox.com/docs/tutorials/curriculums/user-interface-design)

   这个更贴近你的目标。重点看：
   - [Wireframe your layouts](https://create.roblox.com/docs/tutorials/curriculums/user-interface-design/wireframe-your-layouts)
   - [Implement designs in Studio](https://create.roblox.com/docs/tutorials/curriculums/user-interface-design/implement-designs-in-studio)

   这部分会帮你理解：UI 不只是好看，还要适配 Roblox 的玩家操作、设备尺寸、核心 UI、安全区域。

5. **Roblox DevForum UI 设计指南**
   [The Ultimate UI Design Guide](https://devforum.roblox.com/t/the-ultimate-ui-design-guide/1236916)

   这个偏 Roblox 社区实践，适合看别人怎么从草图、Figma 到 Roblox 展示 UI。不要照单全收，重点学思路。

**国内教程怎么选**

B 站可以搜这些关键词：

```text
Figma 入门 UI 设计 Auto Layout
Figma 组件 教程
Figma 导出 PNG 教程
Roblox UI Figma 导入
Roblox Studio GUI 教程
```

我不建议你一开始看那种“UI/UX 全流程 20 小时大课”。你现在只需要会做 Roblox 用的素材，所以优先找标题包含这些词的课：

- `Figma 零基础`
- `Auto Layout`
- `组件`
- `导出 PNG`
- `游戏 UI`
- `Roblox GUI`

**Roblox + Figma 插件可作为后期工具**

等你手动导入熟练后，可以了解：

- [Figma Importer Assistant](https://devforum.roblox.com/t/v120-figma-importer-assistant-a-faster-way-to-create-your-ui/2961561)
- [FigBloxUI](https://figbloxui.dev/)

但我不建议你一开始依赖插件。先手动做 3 到 5 个小 UI，理解 Roblox 的 `Frame / ImageLabel / TextLabel / ImageButton / Scale / AnchorPoint / UIAspectRatioConstraint`，再用插件提速。

**你的最短学习路线**

```text
第 1 天：Figma 基础操作、Frame、Shape、Text、图层
第 2 天：按钮、面板、描边、阴影、Auto Layout
第 3 天：导出 PNG、透明背景、图标/发光层拆分
第 4 天：Roblox Studio 中用 ImageLabel / TextLabel 还原
第 5 天：做一个完整小 UI：货币栏 + 按钮 + 搜索框
```

最适合你的结论是：

> 官方 Figma 入门 + 官方 Auto Layout + Roblox 官方 UI Curriculum，再配合你刚看的 Kek 视频。不要先学完整设计理论，先做 Roblox 能用的 UI 素材。
