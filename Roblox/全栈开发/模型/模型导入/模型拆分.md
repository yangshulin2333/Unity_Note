## 🚀 Roblox 模型多贴图导入解决方案

### 1. 核心问题定位

- **现象：** 模型在 3ds Max 里显示正常，但导出到 Roblox 后发现只能贴一张 PNG，其他部分变白或贴图错乱。
    
- **原因：** Roblox 的 `MeshPart` 零件目前仅支持单个 `TextureID`。如果你的模型在建模软件中使用了多个材质球（对应多张贴图），Roblox 无法在一个零件上同时显示它们。
    

### 2. 解决方案：按材质 ID 强行拆件

通过 3ds Max 的“分离”功能，将共用同一张贴图的零件拆分为独立的物体。

#### 第一步：进入元素编辑模式

1. 选中模型，在右侧 **修改 (Modify)** 面板中点击 **红色小正方体图标**（元素/Element 层级，快捷键 `5`）。
    
2. 此时模型应处于可编辑状态。
    

#### 第二步：按材质 ID 自动选择

1. 在右侧面板向下滚动，找到 **多边形：材质 ID (Polygon: Material IDs)** 栏目。
    
2. 在输入框输入 `1`，点击 **选择 ID (Select ID)**。此时，使用第 1 张贴图的所有零件会变红。
    
    - _注：如果 ID 1 没反应，就试 ID 2、3... 直到选到你想要的部件。_
        

#### 第三步：执行分离 (Detach)

1. 找到 **编辑几何体 (Edit Geometry)** 下方的 **分离 (Detach)** 按钮。
    
2. **关键设置：** 在弹出的窗口中，**不要勾选**“分离到元素”和“以克隆对象分离”。
    
3. 点击 **确定**。刚才红色的部分现在已成为独立物体。
    

#### 第四步：重置轴心（防止对齐偏移）

1. 退出元素模式（点击红色小方块使其不再高亮）。
    
2. 选中拆出来的零件，去右侧 **层次 (Hierarchy)** 面板（第三个图标）。
    
3. 点击 **仅影响轴 (Affect Pivot Only)** -> 点击 **居中到对象 (Center to Object)**。
    
4. 关闭“仅影响轴”，并将该物体的世界坐标（屏幕最下方 X, Y, Z）手动改为 `0, 0, 0`。
    

### 3. 导出与导入流程

1. **导出：** 选中拆好的单个物体，点击 **文件 (File) -> 导出 (Export) -> 导出选定对象 (Export Selected)**，格式选 **FBX**。
    
2. **重复：** 将 5 张贴图对应的部件分别导出为 5 个 FBX 文件。
    
3. **Roblox 导入：**
    
    - 将 5 个 FBX 全部导入 Roblox Studio。
        
    - 在每个 `MeshPart` 的 `TextureID` 栏里，分别上传对应的 PNG 贴图。
        
    - **快速拼装：** 将这 5 个零件的 `Position` 属性全部设为相同（例如 `0, 50, 0`），它们就会像积木一样完美拼合。
        

---

### 💡 专家提醒

- **重复件复用：** 如果模型里有两个一模一样的柱子，只需导出一个。在 Roblox 里 `Ctrl+D` 复制一份即可，这样能极大节省游戏的加载内存。
    
- **炸开命令：** 如果点击模型发现是整体但点不开，先去顶部菜单 **组 (Group) -> 炸开 (Explode)**，直到能选到基础零件为止。
