| 分支父类           | 核心能力                      | 代表子类                                                   |
| -------------- | ------------------------- | ------------------------------------------------------ |
| `PVInstance`   | 3D 世界可放置对象（带位置 / 旋转 / 碰撞） | `Part`/`Model`/`Tool`/`Character`                      |
| `GuiObject`    | 2D UI 界面对象（屏幕显示 / 交互）     | `Frame`/`ImageButton`/`TextLabel`/`ScreenGui`          |
| `BackpackItem` | 可放入玩家背包的物品                | `Tool`（唯一子类）                                           |
| `Humanoid`     | 角色控制器（生命 / 移动 / 动画）       | 直接子类：`Player`的角色控制器                                    |
| `Service`      | 全局服务容器（游戏管理 / 数据）         | `StarterPack`/`Backpack`/`ReplicatedStorage`/`Players` |
