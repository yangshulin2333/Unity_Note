## 完整的 Instance 核心子类架构

| 分支父类                  | 核心能力                    | 代表子类                                                   | 常见实例        |
| --------------------- | ----------------------- | ------------------------------------------------------ | ----------- |
| `PVInstance`          | 3D 世界可放置对象（带位置/旋转/碰撞）   | `Part`/`Model`/`Tool`/`Character`                      | 方块、模型、玩家角色  |
| `GuiObject`           | 2D UI 界面对象（屏幕显示/交互）     | `Frame`/`ImageButton`/`TextLabel`/`ScreenGui`          | 按钮、文本框、面板   |
| `BackpackItem`        | 可放入玩家背包的物品              | `Tool`（唯一子类）                                           | 剑、枪、药水      |
| `Humanoid`            | 角色控制器（生命/移动/动画）         | 直接子类：Player的角色控制器                                      | 玩家的血量、跳跃能力  |
| `Service`             | 全局服务容器（游戏管理/数据）         | `StarterPack`/`Backpack`/`ReplicatedStorage`/`Players` | 背包服务、玩家服务   |
| **`BasePart`**        | **物理零件基类（有形状/材质/物理属性）** | `Part`/`WedgePart`/`CylinderPart`/`TrussPart`          | 各种形状的零件     |
| **`ProximityPrompt`** | **玩家交互触发器（靠近按E触发）**     | 无子类（直接使用）                                              | 开门、拾取物品的感应器 |
| **`Decal`**           | **贴花（贴在物体表面的图片）**       | 无子类                                                    | 墙上的涂鸦、地面的标志 |
| **`Texture`**         | **纹理（重复贴图覆盖表面）**        | 无子类                                                    | 砖墙纹理、草地纹理   |
| **`Script`**          | **代码容器（Lua脚本）**         | `LocalScript`/`ModuleScript`                           | 游戏逻辑脚本      |
| **`Sound`**           | **音频播放器**               | 无子类                                                    | 背景音乐、音效     |
| **`Animation`**       | **动画数据容器**              | 无子类                                                    | 角色待机动画、攻击动画 |
| **`Camera`**          | **摄像机（视角控制）**           | 无子类                                                    | 第一人称、第三人称视角 |
| **`Light`**           | **光源（照明系统）**            | `PointLight`/`SpotLight`/`SurfaceLight`                | 手电筒、路灯      |
| **`ParticleEmitter`** | **粒子发射器（特效）**           | 无子类                                                    | 火焰、烟雾、火花    |
| **`Attachment`**      | **附着点（挂载其他对象的位置）**      | 无子类                                                    | 武器挂点、特效挂点   |
| **`Weld`**            | **焊接（连接两个零件）**          | `WeldConstraint`                                       | 固定两个方块在一起   |
| **`SelectionBox`**    | **选择框（高亮显示物体）**         | 无子类                                                    | 鼠标选中高亮      |
| **`BillboardGui`**    | **广告牌UI（面向玩家的浮动UI）**    | `GuiObject`的子类                                         | 血条、名字标签     |
| **`SurfaceGui`**      | **表面UI（贴在物体表面的UI）**     | `GuiObject`的子类                                         | 电脑屏幕显示、触摸屏  |

## 按功能分类的完整架构

```
Instance (根类)
├── 🧱 物理/世界对象
│   ├── PVInstance (3D放置物)
│   │   ├── Part (基础零件)
│   │   ├── Model (模型容器)
│   │   ├── Tool (工具)
│   │   └── Character (角色)
│   ├── BasePart (物理零件基类)
│   │   ├── Part (方块)
│   │   ├── WedgePart (楔形)
│   │   ├── CylinderPart (圆柱)
│   │   └── TrussPart (桁架)
│   └── Terrain (地形)
│
├── 🎨 视觉/特效
│   ├── Decal (贴花)
│   ├── Texture (纹理)
│   ├── Light (光源)
│   ├── ParticleEmitter (粒子特效)
│   ├── SelectionBox (选择框)
│   └── SelectionSphere (球形选择框)
│
├── 🖥️ UI/界面
│   ├── GuiObject (2D UI)
│   │   ├── Frame (框架)
│   │   ├── ImageButton (图片按钮)
│   │   ├── TextLabel (文字标签)
│   │   ├── TextButton (文字按钮)
│   │   ├── TextBox (输入框)
│   │   └── ScreenGui (屏幕界面)
│   ├── BillboardGui (广告牌UI)
│   └── SurfaceGui (表面UI)
│
├── 🎮 游戏逻辑
│   ├── Script (服务器脚本)
│   ├── LocalScript (客户端脚本)
│   ├── ModuleScript (模块脚本)
│   └── ProximityPrompt (交互触发器)
│
├── 🎵 音频/动画
│   ├── Sound (音频)
│   ├── Animation (动画数据)
│   └── AnimationController (动画控制器)

├── 👤 角色系统

│   ├── Weld (焊接)

│   ├── HingeConstraint (铰链)
│   └── SpringConstraint (弹簧)
│
└── 📷 其他
    ├── Camera (摄像机)
    ├── Attachment (附着点)
    ├── Folder (文件夹)
    ├── BoolValue (布尔值容器)
    ├── NumberValue (数字容器)
    └── StringValue (字符串容器)
```
