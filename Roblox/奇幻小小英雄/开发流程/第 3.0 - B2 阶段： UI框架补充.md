```text
Label         = TextLabel
Icon          = ImageLabel

Window        = Frame
Header        = Frame
Content       = Frame / ScrollingFrame
Footer        = Frame
```

# 三、MainHUD 每个对象是干嘛的

## `MainHUD` = ScreenGui

属性建议：

```text
Enabled = true
ResetOnSpawn = false
IgnoreGuiInset = true
```

作用：  
这是整个主界面的容器。

---

## `Root` = Frame

属性建议：

```text
Size = {1, 0}, {1, 0}
Position = {0, 0}, {0, 0}
BackgroundTransparency = 1
Visible = true
```

作用：  
全屏根容器。以后所有主界面 UI 都放它下面。

---

## `TopBar` = Frame

属性建议：

```text
AnchorPoint = 0, 0
Position = {0, 0}, {0, 0}
Size = {1, 0}, {0, 100}
BackgroundTransparency = 1
```

作用：  
顶部区域，放玩家信息、资源、签到、充值等。

---

## `PlayerInfoPanel` = Frame

属性建议：

```text
Position = {0, 20}, {0, 10}
Size = {0, 220}, {0, 80}
BackgroundTransparency = 0.3
```

里面放：

```text
AvatarImage  = ImageLabel
NameLabel    = TextLabel
LevelLabel   = TextLabel
```

---

## `CurrencyPanel` = Frame

属性建议：

```text
Position = {0, 250}, {0, 10}
Size = {0, 360}, {0, 80}
BackgroundTransparency = 0.3
```

里面放三个资源项：

```text
GoldItem   = Frame
WoodItem   = Frame
StoneItem  = Frame
```

每个资源项内部：

```text
Icon        = ImageLabel
AmountLabel = TextLabel
```

---

## `TopButtons` = Frame

属性建议：

```text
AnchorPoint = 1, 0
Position = {1, -20}, {0, 10}
Size = {0, 300}, {0, 80}
BackgroundTransparency = 1
```

里面放：

```text
SignInButton   = TextButton
RechargeButton = TextButton
MailButton     = TextButton
```

---

## `RightMenu` = Frame

属性建议：

```text
AnchorPoint = 1, 0.5
Position = {1, -20}, {0.5, 0}
Size = {0, 120}, {0, 300}
BackgroundTransparency = 1
```

里面放：

```text
FriendButton   = TextButton
QuestButton    = TextButton
SettingsButton = TextButton
```

---

## `BottomMenu` = Frame

属性建议：

```text
AnchorPoint = 0.5, 1
Position = {0.5, 0}, {1, -20}
Size = {0, 500}, {0, 90}
BackgroundTransparency = 0.2
```

里面放：

```text
ShopButton      = TextButton
InventoryButton = TextButton
DungeonButton   = TextButton
```

# 四、功能弹窗统一对象类型

比如 `SignInGui`、`RechargeGui`、`FriendGui`、`ShopGui`、`InventoryGui` 都用同一套结构。

以 `SignInGui` 为例：

```text
StarterGui
└─ SignInGui               ScreenGui
   └─ Root                 Frame
      ├─ BackgroundDim     Frame
      └─ Window            Frame
         ├─ Header         Frame
         │  ├─ TitleLabel  TextLabel
         │  └─ CloseButton TextButton
         ├─ Content        Frame / ScrollingFrame
         └─ Footer         Frame
            └─ ConfirmButton TextButton，可选
```

---

# 五、弹窗对象属性建议

## `SignInGui` / `RechargeGui` / `FriendGui` = ScreenGui

属性：

```text
Enabled = true
ResetOnSpawn = false
IgnoreGuiInset = true
```

注意：  
不要用 `Enabled = false` 控制显示。  
我们后面统一控制 `Root.Visible`。

---

## `Root` = Frame

属性：

```text
Size = {1, 0}, {1, 0}
Position = {0, 0}, {0, 0}
BackgroundTransparency = 1
Visible = false
```

作用：  
整个弹窗的总容器。默认隐藏。

---

## `BackgroundDim` = Frame

属性：

```text
Size = {1, 0}, {1, 0}
Position = {0, 0}, {0, 0}
BackgroundColor3 = 黑色
BackgroundTransparency = 0.4
```

作用：  
弹窗打开时，背景变暗。

---

## `Window` = Frame

属性：

```text
AnchorPoint = 0.5, 0.5
Position = {0.5, 0}, {0.5, 0}
Size = {0, 600}, {0, 420}
BackgroundTransparency = 0
```

作用：  
弹窗主体。

---

## `Header` = Frame

属性：

```text
Size = {1, 0}, {0, 60}
Position = {0, 0}, {0, 0}
```

作用：  
弹窗顶部标题栏。

里面：

```text
TitleLabel  = TextLabel
CloseButton = TextButton
```

---

## `TitleLabel` = TextLabel

属性建议：

```text
Text = "签到" / "充值" / "好友"
Size = {1, -80}, {1, 0}
Position = {0, 20}, {0, 0}
TextScaled = true
BackgroundTransparency = 1
```

---

## `CloseButton` = TextButton

属性建议：

```text
Text = "X"
AnchorPoint = 1, 0.5
Position = {1, -20}, {0.5, 0}
Size = {0, 40}, {0, 40}
```

---

## `Content` = Frame 或 ScrollingFrame

### 简单界面用 Frame

比如充值：

```text
Content = Frame
```

### 列表界面用 ScrollingFrame

比如好友列表、邮件列表：

```text
Content = ScrollingFrame
```

属性建议：

```text
Position = {0, 0}, {0, 60}
Size = {1, 0}, {1, -120}
```

---

## `Footer` = Frame

属性：

```text
AnchorPoint = 0, 1
Position = {0, 0}, {1, 0}
Size = {1, 0}, {0, 60}
```

作用：  
底部按钮区域。比如确认、领取、购买等。

# 七、CommonGui 是什么类型

`CommonGui` 也是 `ScreenGui`。

推荐结构：

```text
CommonGui                  ScreenGui
└─ Root                    Frame
   ├─ ToastLabel           TextLabel
   └─ ConfirmDialog        Frame
      ├─ MessageLabel      TextLabel
      ├─ ConfirmButton     TextButton
      └─ CancelButton      TextButton
```

前期你可以先不做 `CommonGui`，或者只建空壳。

---

# 八、ShopGui 和 InventoryGui 暂时怎么处理

你已经有 `ShopGui` 和 `InventoryGui`，建议也改成统一弹窗模板：

```text
ShopGui                    ScreenGui
└─ Root                    Frame
   ├─ BackgroundDim        Frame
   └─ Window               Frame
      ├─ Header
      │  ├─ TitleLabel
      │  └─ CloseButton
      ├─ Content
      └─ Footer
```

```text
InventoryGui               ScreenGui
└─ Root                    Frame
   ├─ BackgroundDim        Frame
   └─ Window               Frame
      ├─ Header
      │  ├─ TitleLabel
      │  └─ CloseButton
      ├─ Content
      └─ Footer
```

# 九、你现在最该先建哪些

不要一次把全部做完。  
你先建这几个：

```text
MainHUD
SignInGui
RechargeGui
FriendGui
ShopGui
InventoryGui
```

其中优先顺序：

1. `MainHUD` 重构
    
2. `SignInGui`
    
3. `RechargeGui`
    
4. `FriendGui`
    
5. `ShopGui`
    
6. `InventoryGui`
